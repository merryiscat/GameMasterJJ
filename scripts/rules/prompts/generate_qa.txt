당신은 보드게임 전문가입니다.
아래 보드게임 룰 섹션을 읽고, 실제 사용자가 물어볼 만한 Q&A 쌍을 생성하세요.

## Q&A 작성 규칙

1. 실제 보드게임 플레이 중에 물어볼 법한 자연스러운 질문을 만드세요
2. 질문은 구어체로 작성하세요 (예: "몇 명까지 할 수 있어?", "내 턴에 뭘 할 수 있어?")
3. 답변은 정확하고 간결하게 작성하세요
4. 섹션 내용에 따라 3~10개 Q&A를 생성하세요 (내용이 많으면 더 많이)
5. 다양한 난이도의 질문을 포함하세요 (기초 ~ 심화)

## 출력 형식

반드시 "qa_pairs" 키 안에 JSON 배열로 출력하세요.
최소 3개 이상의 Q&A 쌍을 생성하세요.

```json
{"qa_pairs": [
  {
    "question": "질문",
    "answer": "답변"
  }
]}
```

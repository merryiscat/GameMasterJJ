## 정리 규칙

1. 원문의 핵심 정보를 빠짐없이 보존하세요.
2. 모호하거나 불완전한 표현은 명확하게 다듬으세요.
3. 마크다운 서식(제목, 리스트, 표)을 활용하여 읽기 쉽게 구조화하세요.
4. 구조화 가능한 섹션(setup_by_player, actions, keywords)은 항목별로 분리하세요.

## 출력 형식

JSON으로 출력하세요:

```json
{
  "cleaned": "정리된 전체 텍스트 (마크다운)",
  "items": ["항목1", "항목2", ...]
}
```

- `cleaned`: 정리된 섹션 전체 텍스트
- `items`: 구조화 가능한 섹션만 항목별 분리 (그 외 섹션은 빈 배열 [])
  - setup_by_player → 인원수별로 분리 (예: ["2인: ...", "3인: ...", "4인: ..."])
  - actions → 액션 종류별로 분리 (예: ["건설 액션: ...", "대출 액션: ..."])
  - keywords → 키워드별로 분리 (예: ["보안관: ...", "무법자: ..."])
